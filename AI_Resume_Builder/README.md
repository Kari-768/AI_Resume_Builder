# AI-Based Resume Builder

A Python Flask web application that converts resumes of any format
(PDF, DOCX, TXT) into a clean, resume format.

## Features
- Upload resume in PDF / DOCX / TXT
- Automatic section detection
- Universal ATS-friendly resume output
- Downloadable resume

## Tech Stack
- Python
- Flask
- HTML & CSS
- pdfplumber
- python-docx

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000/
